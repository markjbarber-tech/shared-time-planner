# Project Memory

## Core
Architectural Vellum Grid aesthetic. Cormorant Garamond + Space Grotesk fonts.
Supabase via Lovable Cloud. Published at time-together-share.lovable.app.
Anonymous-first: localStorage until user signs up.
GitHub sync: markjbarber-tech/sharedcalendarapp.git.
